despasito.equations_of_state.saft.gamma_mie_funcs.calc_Aideal¶

despasito.equations_of_state.saft.gamma_mie_funcs.calc_Aideal(xi, rho, massi, T)[source]¶

Return a vector of ideal contribution of Helmholtz energy. \(\frac{A^{ideal}}{N k_{B} T}\)

Parameters

  • xi (numpy.ndarray) – Mole fraction of each component, sum(xi) should equal 1.0

  • rho (numpy.ndarray) – Number density of system [molecules/m^3]

  • massi (numpy.ndarray) – Mass for each component [kg/mol]

  • T (float) – Temperature of the system [K]

Returns

Aideal – Helmholtz energy of ideal gas for each density given.

Return type

nd.array
