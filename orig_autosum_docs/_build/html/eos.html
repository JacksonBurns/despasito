Equations of State¶

Here we list list the available equations of state and the function used to access them.

equations_of_state

This vision for this library is that this file will define an EOS class, and import sub-libraries of each EOS family

Available EOS¶

equations_of_state.saft.gamma_mie

EOS object for SAFT-\(\gamma\)-Mie Equations referenced in this code are from V.

equations_of_state.saft.gamma_mie_funcs

Routines for calculating the Helmholtz energy for the SAFT-gamma equation of state.

equations_of_state.saft.nojit_exts

Routines for calculating the Helmholtz energy for the SAFT-gamma equation of state.

equations_of_state.saft.jit_exts

Adding an EOS¶

Adding an EOS family is easily implemented by adding a new directory to the equations_of_state directory. A new EOS is then added by adding a module with the desired EOS inside that is derived from our EOS interface (shown below).

Note

In the future, a USER folder will be included in our program so that users can collect all of their personal additions and modifications to DESPASITO in one location.

equations_of_state.interface

This file contains the interface needed to create further equation of state (EOS) objects.
