Adding New Data Objects¶

New data objects should be based on and reference the experimental data abstract class. Although this template descibes at a high level what is necessary, see existing examples for more details.

class despasito.fit_parameters.interface.ExpDataTemplate[source]¶

Interface needed to create further objects to represent experimental data. Using this template all future data types will be easily exchanged.
